Welcome to the Instagram account green score application!
This application takes an Instagram account username and returns a graphic with the most recent “green” post with the username and green score printed on top of it.
To run it, the following libraries are required:
random
PIL
time
json
requests
bs4
google.cloud
google.oauth2
io
sys

Moreover, a .json file with the google vision cloud credentials is needed, and it is provided in the zip as patyhtjemet.json . This particular credentials file has a limit of 1200 pictures, to get more a new credential file would have to be obtained.

Moreover, the font file for the particular font we used is also necessary to be in the directory. It is also included in the zip file.

Finally, to get the program running, simply run the code and follow the instructions in the console.
